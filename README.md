Basic Commands
--------------

* Build and run
~~~~~~~~~~~~~~~~~~~~~~~~~~
    $ docker-compose build
    $ docker-compose up
~~~~~~~~~~~~~~~~~~~~~~~~~~

* Run shell
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    $ docker-compose run django sh 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

If you want to run a project without docker set **READ_DOT_ENV_FILE=true** and specify envs in **src/.env** file.
